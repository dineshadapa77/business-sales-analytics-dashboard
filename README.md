# Business Sales Analytics Dashboard

This project analyzes retail sales performance using SQL and Power BI.

## Project Overview
The goal of this project is to transform raw transactional data into business insights by building a data warehouse and an interactive dashboard.

## Technologies Used
- SQL
- Power BI
- DAX
- Data Modeling

## Data Model
A star schema was designed with:
- fact_sales
- dim_product
- dim_customer
- dim_date
- dim_region
- dim_channel
- fact_targets_monthly

## Key Metrics
- Revenue
- Profit
- Margin %
- Orders
- Average Order Value
- Return Rate
- Revenue Variance
- Revenue Attainment %

## Dashboard Pages
1. Executive Summary
2. Product Performance
3. Region & Channel Analysis
4. Customer Analysis

## Insights
- Identified regions underperforming against targets
- Found products with high revenue but low margin
- Analyzed customer segment contribution
- Evaluated impact of discounts on profitability
